# Countdown Challenge

A simple, local-first web application for timing challenges.

## Deployment to Vercel

This project is ready to be deployed to [Vercel](https://vercel.com).

### Option 1: Drag and Drop (Easiest)
1. Go to your [Vercel Dashboard](https://vercel.com/dashboard).
2. Click "Add New..." -> "Project".
3. Drag the entire `countdown` folder into the import area.
4. Follow the prompts to deploy.

### Option 2: Git Integration (Recommended)
1. Initialize a git repository in this folder if you haven't already.
2. Push your code to GitHub, GitLab, or Bitbucket.
3. Import the repository in Vercel.
4. Vercel will automatically detect the static site structure and deploy it.

## Features
- Countdown timer with minutes and seconds input.
- "Urgent mode" visual and audio feedback in the last 10 seconds.
- Custom "Horn" sound effect on completion.
- Dual Gold Eiffel Tower silhouette animation that reveals from bottom to top as time progresses.
